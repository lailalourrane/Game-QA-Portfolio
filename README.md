# Game QA & Exploratory Testing Portfolio

Welcome to my Game Quality Assurance and Exploratory Testing portfolio. This repository serves as a central hub for documenting bugs, interface inconsistencies, and localization issues discovered during my deep-dive testing sessions across various game titles.

My testing methodology blends technical analysis (Game Engines, Physics, and Mechanics) with **UI/UX Heuristics** and **Localization QA (LQA)** to help developers polish user experience, interface elasticity, and digital accessibility.
